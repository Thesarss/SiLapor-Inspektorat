CARA SETUP:
-----------
1. Install dependencies:
   cd backend && npm install
   cd ../frontend && npm install

2. Setup database:
   cd backend
   cp .env.example .env
   # Edit .env dengan database credentials
   npm run migrate

3. Run development:
   # Terminal 1
   cd backend && npm run dev
   
   # Terminal 2
   cd frontend && npm run dev
