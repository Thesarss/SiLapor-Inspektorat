## 📝 Next Steps

### For Collaborators:
1. Clone the repository:
   ```bash
   git clone https://github.com/Thesarss/SiLapor-Inspektorat.git
   cd SiLapor-Inspektorat
   ```

2. Install dependencies:
   ```bash
   # Backend
   cd backend
   npm install
   
   # Frontend
   cd ../frontend
   npm install
   ```

3. Setup environment:
   ```bash
   # Copy .env.example to .env
   cp backend/.env.example backend/.env
   # Edit .env with your database credentials
   ```

4. Run migrations:
   ```bash
   cd backend
   npm run migrate
   ```

5. Start development:
   ```bash
   # Terminal 1 - Backend
   cd backend
   npm run dev
   
   # Terminal 2 - Frontend
   cd frontend
   npm run dev
   ```

### For Deployment:
- See [Deployment Guide](user-guide/deployment/HOSTING_ONLINE_GUIDE.md)
- Configure production environment variables
- Build for production
- Deploy to hosting service
